# APIPROBE

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/apiprobe.git
cd apiprobe
go build -o apiprobe .
```

## Usage

```bash
./apiprobe --help
```
