## Apiprobe

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/apiprobe.git
cd apiprobe
go build -o apiprobe .
```

### Usage

```bash
./apiprobe --help
```

---

## Quick Start

```bash
./apiprobe [command]
```
